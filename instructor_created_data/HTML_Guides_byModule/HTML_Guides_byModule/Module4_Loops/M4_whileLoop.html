while Loop¶

  • Condition tested for true or false value
  • Statements repeated as long as condition remains True.
In [ ]:
score = 3
In [ ]:
while score < 0:
    print('invalid score entered!!!')
In [ ]:
score

In order for a loop to stop executing, something has to happen inside the loop to make the condition false¶

  • If variable score above is assigned a value of -1 , the while loop will iterate (run) infinitely.
  • To prevent an infinite loop, something in the while suite must change score’s value, so the condition eventually becomes False.
