Lists¶

Creating a List¶

In [1]:
c = [-45, 6, 0, 72, 1543]
In [2]:
c
Out[2]:
[-45, 6, 0, 72, 1543]

Accessing Elements of a List¶

  • Reference a list element by writing the list’s name followed by the element’s index enclosed in [] (the subscription operator).

In [3]:
c[0]
Out[3]:
-45
In [4]:
c[4]
Out[4]:
1543

Determining a List’s Length¶

In [5]:
len(c)
Out[5]:
5

Accessing Elements from the End of the List with Negative Indices¶

  • Lists can be accessed from the end by using negative indices:

In [6]:
c[-1]
Out[6]:
1543
In [7]:
c[-5]
Out[7]:
-45

Indices Must be either Integers or variables that reference Integers¶

In [8]:
a = 0
In [9]:
b = 3
In [10]:
c[a]
Out[10]:
-45

Lists Are Mutable (Can be changed)¶

In [11]:
c[4] = 17
In [12]:
c
Out[12]:
[-45, 6, 0, 72, 17]

Appending to a List¶

  • Using with +=
In [13]:
a_list = []
In [14]:
for number in range(1, 6):
    a_list += [number]
In [15]:
a_list
Out[15]:
[1, 2, 3, 4, 5]
  • Using List methods
In [16]:
a_list.append(9) # adds 9 to the end of the list
In [17]:
a_list
Out[17]:
[1, 2, 3, 4, 5, 9]
In [18]:
a_list.extend([45,67]) # adds 45,67 to the end of the list
In [19]:
a_list
Out[19]:
[1, 2, 3, 4, 5, 9, 45, 67]
